# Wallet

to run:
`nx serve`
or if you don't have installed nx yet:
`node_modules/.bin/nx serve`

## Structure
this is standard Angular architecture contains
1. components
2. pages
3. services
4. interfaces

## Routes

1. login 
2. pages/dashboard
3. pages/wallet
